# IoT Intrusion Detection Using AI Techniques

AI-based IoT Intrusion Detection System developed using Machine Learning and Deep Learning techniques to classify malicious network traffic using the CICIoT2023 dataset.

## 🚀 Models Implemented

* Multilayer Perceptron (MLP)
* Random Forest Classifier
* Convolutional Neural Network (CNN)
* K-Means Clustering

## 📊 Dataset

CICIoT2023 Intrusion Detection Dataset
https://www.kaggle.com/datasets/subhajournal/iotintrusion

## 🛠 Technologies Used

* Python
* TensorFlow / Keras
* Scikit-learn
* Pandas & NumPy
* Matplotlib & Seaborn
* Jupyter Notebook

## 📈 Results

| Model         | Accuracy |
| ------------- | -------- |
| MLP           | 77.57%   |
| CNN           | 84.70%   |
| Random Forest | 99.76%   |
